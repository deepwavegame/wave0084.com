# Unity Smart Sheet: Bắt Đầu

**Unity Smart Sheet** là công cụ tối ưu giúp bạn quản lý dữ liệu Game ngay bên trong Unity bằng giao diện dạng bảng (Spreadsheet).

## Cài đặt

1. Mở Unity Editor.
2. Đi tới **Window > Package Manager**.
3. Chọn thẻ **My Assets** và tìm **Unity Smart Sheet**.
4. Nhấn **Download** sau đó **Import** vào project.

## Tính năng chính

- **Giao diện bảng trực quan:** Sửa đổi hàng loạt chỉ số, vật phẩm giống như dùng Excel.
- **Auto-Serialization:** Dữ liệu nhập vào sẽ tự động lưu dưới dạng `ScriptableObject` hoặc `JSON` tuỳ chọn.
- **Google Sheets Sync:** Cập nhật dữ liệu từ xa giúp team Game Designer và Developer làm việc đồng bộ.

## Khởi tạo Bảng đầu tiên

1. Chuột phải trong cửa sổ **Project** -> `Create > Wave0084 > Smart Sheet`.
2. Mở file vừa tạo, một cửa sổ dạng bảng sẽ hiện ra.
3. Bắt đầu định nghĩa các cột (ID, Tên, Sát Thương, Máu...).
4. Dữ liệu sẽ tự động lưu sau mỗi thay đổi.

> **Mẹo:** Bạn có thể kéo thả trực tiếp các Prefab hoặc AudioClip vào các ô của bảng để tham chiếu!
